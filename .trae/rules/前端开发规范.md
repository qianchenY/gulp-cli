# 响应式企业官网 前端开发规范（AI 参考文档）

> 本文档用于 AI 代码生成时的参考，请严格遵循以下规范生成代码。
> 安全宽度：默认 1400px，具体数值以设计图为准（可通过 `$width` 变量配置）
> 所有宽度计算基于 `scss_init.scss` 中的 `$width` 变量进行

---

## 项目概述

- **技术栈**：Gulp + EJS + SCSS + jQuery + Swiper插件
- **设计基准**：默认 1400px 安全宽度（以设计图为准，通过 `$width` 变量调整）
- **语言**：英文站
- **类型**：响应式企业官网

## SCSS 规范

### 变量定义 (scss_init.scss)

```scss
// 安全宽度（默认1400px，具体以设计图为准）
$width: 1400px;

// 颜色
$color: #00A2E9;    // 主色调，具体以设计图为准
$color2: #055DAF;   // 辅助色，具体以设计图为准

// 字体
$font-pr: 'PoppinsRegular', arial, helvetica, sans-serif; // 默认字体，具体以设计图为准
$font-pm: 'PoppinsMedium', arial, helvetica, sans-serif; // 中等粗，具体以设计图为准
$font-psb: 'PoppinsSemiBold', arial, helvetica, sans-serif; // 半粗，具体以设计图为准
```

### 核心函数

```scss
// 百分比宽度计算（基于安全宽度 $width）
@function get_w($w, $max: 1400, $b: 100){
    @return math.div($w, $max) * $b + 0%;
}

// 使用示例
width: get_w(300);           // 300px在安全宽度中的百分比
width: get_w(370, 788, 100); // 基于788px的局部百分比（设计图模块非安全宽度时）

// vw单位计算
@function get_vw($size, $media: 1920) {
    @return math.div($size, $media) * 100vw;
}
```

### 混合宏 (Mixin)

```scss
// 字体大小（同时生成px和rem）
@include font-size(16);  // font-size: 16px; font-size: 1.6rem;

// 外边距
@include margin($top, $right, $bottom, $left);

// 过渡动画
@include transition(all, 500ms);

// 浮动
@include float(left);

// placeholder样式
@include placeholder(#333);
```

### 书写规范

#### 1. 层级嵌套（不超过3级）

```scss
.m-模块名{
    .box-item{
        .item-pic{
            // 最多3级，不要再嵌套
        }
    }
}
```

当前组件的样式必须都在 `.m-` 开头的类名下书写，避免影响其他组件的样式。

#### 2. 布局优先使用 Flex

```scss
.box-wp{
    display: flex;
    flex-flow: row wrap;        // 横排换行
    justify-content: flex-start;  // 左对齐（除非是一行两列布局，否则禁止space-between）
    align-items: flex-start;
    gap: 30px get_w(20);        // 垂直间距固定px，水平间距百分比
}
```

#### 3. 响应式媒体查询

**断点顺序**：从大到小，放在模块最底部

```scss
.m-模块名{
    // 默认样式（桌面端）
    
    @media screen and (max-width: 1440px){
        // 大屏 - 完全还原设计图，设计图有的内容不隐藏，字号、间距、行高1:1匹配
    }
    
    @media screen and (max-width: 991px){
        // 平板端
    }
    
    @media screen and (max-width: 640px){
        // 手机端
    }
    
    @media screen and (max-width: 375px){
        // 小屏手机
    }
}
```

所有宽度、间距、字号等尺寸变化必须基于 `.f-box` 的特性进行等比缩放调整。

#### 4. 宽度计算

所有响应式宽度必须使用 `get_w($w)` 函数，默认基准 `$width`。

**禁止硬编码百分比或固定 px 值。**

`get_w()` 内只放纯数字，禁止传入 `%` 或 `px` 单位。

如果从设计图中获取模块宽度时其并非 `$width`，则在该组件下使用 `get_w($w, $base)` 函数根据指定安全范围换算。

#### 5. 高度计算

默认不为任何元素（`input`、`textarea`、`select` 等表单元素除外）设置固定高度 `height`。

如有需要 `height` 在布局中需要使用固定单位（`px`），并进行响应式适配。

**图片处理**：

```scss
img{
    display: block;
    width: 100%;
    height: auto;  // 自适应高度，不设置固定高度
}

// 懒加载类
.j-lazy{
    // 使用 data-original 属性
}
```

#### 6. 字体大小

必须使用 `@include font-size($value)` 混合宏，自动生成 px 和 rem 值。

**禁止直接使用 `font-size` 属性。**

#### 7. 间距处理

| 间距属性 | 缩放方式 | 说明 |
|----------|----------|------|
| `column-gap` | `get_w()` 百分比 | 水平间距随安全范围换算 |
| `margin-left` | `get_w()` 百分比 | 左外边距 |
| `margin-right` | `get_w()` 百分比 | 右外边距 |
| `row-gap` | 固定 px | 垂直间距不要用百分比 |
| `padding` | 固定 px，响应式调整 | 避免使用 `get_w($w)` 导致间距随容器缩放过于剧烈 |

#### 8. 文本截断

标题/描述等多行文本默认最多显示2行，实际行数参考设计图，按最大行数设置。

```scss
// 多行省略
.box-title{
    display: -webkit-box;
    -webkit-line-clamp: 2;  // 行数
    -webkit-box-orient: vertical;
    overflow: hidden;
    text-overflow: ellipsis;
}
```

## 列表模块开发规范（新闻、产品等列表通用）

### 间距设置

- `column-gap` 使用 `get_w(xx)` 计算水平间距
- `row-gap` 使用固定像素值（默认30px，实际按照设计图调整）
- 垂直间距不要用百分比

### 标准多列布局

```scss
// 列布局（示例：5列）
.box-wp{
    display: flex;
    flex-flow: row wrap;
    justify-content: flex-start;
    gap: 30px get_w(20);  // row-gap固定px，column-gap百分比
}

.box-item{
    width: get_w(300);    // 单列宽度
}

// 响应式调整
@media screen and (max-width: 991px){
    .box-item{
        width: calc(50% - #{get_w(10)});  // 2列
    }
}

@media screen and (max-width: 640px){
    .box-item{
        width: 100%;  // 1列
    }
}
```

### 样式复用

列表项公共样式提取为独立的 `m-` 类（如 `.m-hnewsitem`），方便其他页面复用。

模块内的 `.xx-item` 只保留宽度、外间距等与布局相关的特有样式，公共样式继承自 `m-` 类。

```scss
// 提取公共样式（如：新闻列表项）
.m-hnewsitem{
    .item-title{
        @include font-size(18);
        line-height: 26px;
    }
}

// 页面内只写布局相关
.m-hnews{
    .box-item{
        width: get_w(500);
        margin-bottom: 20px;
    }
}
```

## JavaScript 规范

### 引入方式

- 所有 JavaScript 代码必须在 `/src/js/` 目录下，文件名必须以 `.js` 结尾
- 所有 JavaScript 代码必须在页面底部引入，默认只在当前页面引入
- jQuery 优先引入（jquery-1.11.3.min.js）
- 然后引入第三方插件（swiper.min.js）
- 最后引入业务 JS（main.js, base.js）

### 类名规范

- JS 交互钩子以 `j-` 开头：`.j-nextshow`, `.j-lazy`, `.j-offset-top`
- 禁止使用 CSS 样式类作为 JS 选择器

### Swiper 轮播插件

swiper 为默认轮播图插件，所有轮播图优先使用 swiper 插件。

```html
<div class="swiper mySwiper">
    <div class="swiper-wrapper">
        <div class="swiper-slide">...</div>
    </div>
</div>
```

### 常用交互类

| 类名 | 功能 |
|------|------|
| `.j-nextshow` | 点击显示下一个元素 |
| `.j-lazy` | 图片懒加载 |
| `.j-offset-top/right/bottom/left` | 滚动动画偏移方向 |
| `.j-videoshow` | 视频弹窗 |

## 响应式适配策略

### 断点对照表

| 断点 | 设备 | 适配策略 |
|------|------|----------|
| 1440px+ | 大屏桌面 | 1:1 完全还原设计图，设计图有的内容不隐藏，字号、间距、行高1:1匹配 |
| 1440px | 普通桌面 | 微调间距，缩小字体 |
| 991px | 平板端 | 列数减少（5→2，3→1），导航折叠 |
| 640px | 手机端 | 单列布局，元素居中，字体更小 |
| 375px | 小屏手机端 | 最小尺寸适配，间距紧凑 |

### 尺寸缩放原则

| 属性 | 缩放方式 | 示例 |
|------|----------|------|
| width | `get_w()` 百分比 | `width: get_w(300)` |
| column-gap | `get_w()` 百分比 | `column-gap: get_w(20)` |
| row-gap | 固定 px | `row-gap: 30px` |
| padding | 固定 px，响应式调整 | `padding: 50px 20px` |
| font-size | `@include font-size()` | `@include font-size(16)` |

## 最佳实践

### 1. 命名语义化
- ✅ `.m-hnews`（home-news）
- ❌ `.m-news1`

### 2. 避免硬编码
- ✅ `width: get_w(300)`
- ❌ `width: 18.75%` 或 `width: 300px`

### 3. 层级控制
- ✅ 3级以内嵌套
- ❌ 深层次嵌套导致权重过高

### 4. 媒体查询位置
- ✅ 每个模块底部写自己的媒体查询，按断点从大到小分组
- ❌ 所有媒体查询集中在文件最后

### 5. Flex 布局
- ✅ `justify-content: flex-start`
- ❌ `space-between`（最后一行对齐问题，除非是一行两列的布局）
